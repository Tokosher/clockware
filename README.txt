План по разработке

Клиентская часть:
1. Создать форму для заказа мастера и связать это с бэкендом
2. В форме города берутся из динамического .json файла в соответствии с городом и свободным временем мастера
3. После заказа последует выбор мастера, который так же берётся из динамического .json файла
4. После заказа мастера отправляется письмо с подтверждением заказа мастера
5. После подтверждения с почты страница сообщает о том, что мастер заказан, благодарим за сотрудничество
6. Общее оформление и дизайн всей части по фронту

Админская часть:
1. Возможность добавлять мастеров
2. Возможность добавлять города
3. Отображение страницы с заказами
4. Добавление возможности редактирования заказа, а именно: удаление заказа, изменение время заказа, мастера и других сущностей из формы заказа
5. Возможность удаления мастера
6. Доступ по определённому логину

Всем юзерам будут назначены уникальные айди, с помощью которых можно определить залогинился ли админ в систему (реализация будет через куки)


Для запуская проекта достаточно использовать npm run dev, если используется node package manager.
